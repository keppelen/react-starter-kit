# Node React Stack
Example project with React + React to String + Enzyme + Mocha

- Node/Express
- React + React Router + Redux
- SASS with Node sass middleware
- Babel 6
- Webpack
- Unit tests (Enzyme + Mocha + Chai + Sinon)

## How to use
```shel
npm intall
npm run dev
```

## In production
```shell
npm run build
npm start
```

## Run tests
```shell
npm test
```
